# DFA Graphizer

## 概要

本プログラムは、DFAを描画する課題を解くために作成されました。

現在は開発版のため、いくつかの変数の値がハードコーディングされています。
汎用的に使用できるように、コマンドライン引数やtxtファイルから値を入力できるようにする予定です。

私はオートマトンの初学者です。
間違いや不適切な用語の使用がありましたら、コメントいただけると幸いです。

## 環境・使用ツール

g++ (Homebrew GCC 14.2.0) 14.2.0
